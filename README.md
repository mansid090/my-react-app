Counter and Form App

A simple React application built with Vite, Tailwind CSS, and React Router to practice core React concepts like state management, routing, reusable components, and forms.

Home Page:
    Link to Counter and Form Page


Counter App:
    A simple react-page with Increment, decrement, and reset counter buttons 

Form Page:
    Simple form with name , email , phone number


Routing:
    Implemented using react-router-dom
        Routes:
            / → Home
            /counter → Counter
            /forms → Form
